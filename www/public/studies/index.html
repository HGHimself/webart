A web study is small internet-based application that demonstrates only a small artistic or technical concept.

The ones you find here are developed with various tools, but the common purpose is to get a feel for building with minimalism.

For the framework, preact is used to manage state and render a self made library of reusable components. Webpack is used to compile this code into the browser-preferred javascript. The end result is a very small bundle size which makes for pleasant render times and a responsize UI

To implement the studies' core concepts, either d3.js or a custom web-assembly module is used. React is used to manage the state and updating of the page, d3 is used for any interesting drawing and rendering of shapes, and web-assembly is used for any interesting computation that would require more complex logic than I'm comfortable using js with.
